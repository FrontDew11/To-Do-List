
# 📝 Python To-Do List App

A simple and efficient command-line application to manage your daily tasks. Built with **Python**, this tool allows you to stay organized directly from your terminal or **Termux**.

## ✨ Features
- **Add Tasks:** Quickly insert new items into your list.
- **Delete by Number:** Remove completed tasks using their index (no need to retype the whole task name).
- **Numbered Overview:** View all your tasks in a clear, numbered format using Python's `enumerate`.
- **Error Handling:** Robust protection against invalid inputs (e.g., entering letters instead of numbers when deleting).

## 🚀 Installation & Usage

To run this app on your machine or in **Termux**, follow these steps:
